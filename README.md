这是我的第一个Python小项目
首先是采用ImageGrab库的函数来捕获图像，然后通过pytesseract这个软件提供的Python接口来进行图片识别和提取文本，再将得到的字符串转化成数字进行比较。最后采用pyautogui库的一些函数来进行鼠标操作。
遇到的第一个问题是导入模块，首先在cmd中执行 pip install modulename 来下载目标模块，再在Pycharm中import modulename。其次，这样的模块是存在于系统Python解释器里的，所以要求Pycharm使用系统Python解释器。而我的Pycharm默认采用了虚拟环境中的解释器，所以还是会找不到模块。（正确形式如图）
![image](https://github.com/user-attachments/assets/9eb329db-8d66-41cd-9a91-87869325d816)
第二个问题是不晓得各个包及其函数的含义和作用，不过好在有ChatGPT,解释的很详尽。
第三个问题是这个项目的不足之处。第一点是，项目采用图像捕获和识别转化，在效率上有很大限制；第二点是，通过pyautogui模块来进行鼠标操作效果很差，导致这个项目一般情况下可以正确识别并判断，但不能在屏幕上用鼠标做出反馈。也做了这个项目的同学可能回避了这个问题，他是采用模拟器快捷键功能来做出反馈；第三个点是，图像捕获需要手动测量px值，很不智能。不过Python我学的不是很多，这个项目也就做了一下午，有这样的效果还是能接受的。
再说说对Python的认识。我最近才学的Python，理论知识和项目经验都很少，但觉得Python语法真的很简单，没有太多条条框框，使用Python编程也很便捷。
暂时就这样，以后掌握了更多知识可能会来完善这个项目。
